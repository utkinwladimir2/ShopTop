# ShopTop / Social Mall

Статический сайт для GitHub Pages: социальная сеть-витрина реферальных подборок в стиле виртуального торгового центра.

## Как включить сайт

1. Откройте репозиторий `ShopTop`.
2. Перейдите в `Settings` → `Pages`.
3. В `Build and deployment` выберите:
   - Source: `Deploy from a branch`
   - Branch: `main`
   - Folder: `/root`
4. Сохраните настройки и дождитесь публикации Pages.

## Как заполнить первую публичную витрину

1. Откройте опубликованный сайт.
2. Перейдите на страницу `#/setup`.
3. Введите название витрины, описание и до 5 товаров с реферальными ссылками.
4. Нажмите `Сохранить в браузере`, чтобы сразу посмотреть витрину у себя.
5. Нажмите `Сгенерировать код`.
6. Скопируйте код и замените им содержимое файла `site-data.js` в GitHub.
7. Сохраните commit. После обновления GitHub Pages эта витрина станет первой публичной витриной для всех посетителей.

## Почему так

GitHub Pages размещает только статические файлы. Он не запускает backend и не может сам записывать данные из формы в репозиторий. Поэтому интерактивные изменения сохраняются в браузере через `localStorage`, а публичная витрина для всех посетителей задается через файл `site-data.js`.

## Что работает

- Главная страница и лента витрин.
- Виртуальный торговый центр.
- Публичные витрины.
- Карточки товаров с реферальной пометкой.
- Лайки, комментарии, подписки.
- Личный кабинет.
- Страница `#/setup` для заполнения первой витрины.
- Коллекция товаров до 100 карточек.
- Выбор 5 активных товаров.
- Редактор витрины: фон, подсветка, декор.
- Внутренние монеты.
- Страница `Мой заработок` с кликами и расчетным доходом.
- Адаптивная темная верстка с неоновыми акцентами.
